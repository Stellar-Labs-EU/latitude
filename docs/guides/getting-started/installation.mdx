## Requirements

To use `latitude-cli`, you need to have Node.js installed on your machine. You can download it from the [official website](https://nodejs.org/).

## Installation

Run the following command to install `latitude-cli` globally:

<CodeGroup>
```bash npm
npm install -g latitude-cli
```
```bash yarn
yarn global add latitude-cli
```
```bash pnpm
pnpm add -g latitude-cli
```
</CodeGroup>
